# AngularJS 1 Tutorial

En læringsapp for AngularJS 1.0.8 med eksempler på:

1. **Data Binding** – vise scope-variabler i view
2. **Two-Way Binding** – input-felt bundet til modell
3. **Select / Cascading** – ng-options med avhengige selects
4. **Search (filter)** – filtrere lister med ng-repeat filter
5. **ng-show / ng-hide** – vise/skjule elementer basert på boolean
6. **Events** – legge til nye personer med ng-click

## Kjøre

Åpne `public_html/index.html` direkte i en nettleser.
